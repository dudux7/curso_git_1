$ git status : ver status do  repositório
$ git init : cria um repositório
$ git add : add arquivos para repositório git
$ git push -u origin main: envia arquivos para git hub
$ git commit -a: salva alteraçoes no git
$ git rm : deleta arquivos
$ git branch : visualiza as branch
$ git branch */ nome da branch /* : cria uma nova branch
$ git branch -d */ nome da branch /* : deleta o nome da branch
$ git checkout : muda de branch
$ git checkout -b : cria e muda de branch
$ git stash : funciona como uma lixeira, ele desfaz e armazena todo codigo não comitado
$ git stash list : exibe as stash
$ git stash apply + num da stash : recupera stash
$ git stash show -p + num da stash : mostra oq tem na stash
$ git stach drop + num da stash : apaga apenas a stach desejada
$ git stach clear: apaga todas as stash
